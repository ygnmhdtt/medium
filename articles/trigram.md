n-gramの勉強のために、簡単に作ってみた。
https://github.com/ygnmhdtt/trigram
ruby製です。

<!--more-->

使い方

```
$ ruby trigram.rb dogura_magura.txt > output.txt
```

例

```
$ ruby trigram.rb test.txt
```

```
test.txt
赤パジャマ青パジャマ黄パジャマ
かえるぴょこぴょこ3ぴょこぴょこ合わせてぴょこぴょこ6ぴょこぴょこ
```
